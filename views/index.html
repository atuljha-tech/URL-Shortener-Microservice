URL Shortener Microservice

API Project: URL Shortener Microservice

Build a full-stack JavaScript app that is functionally similar to this: https://url-shortener-microservice.freecodecamp.rocks

Example Usage:

Example Output:

{
  "original_url": "https://www.example.com",
  "short_url": 1
}

Usage:

  • POST a URL to /api/shorturl and get a JSON response with original_url and short_url properties
  • Visit /api/shorturl/<short_url> to be redirected to the original URL
  • Invalid URLs will return { "error": "invalid url" }
